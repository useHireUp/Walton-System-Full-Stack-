# Assessment Questions

Please answer the following questions as part of your submission. These questions help us understand your development process and decision-making approach.

**Note:** Please keep your answers concise - 1-2 sentences per question is sufficient. The exception is the Decision-Making Question, which should be 2-3 paragraphs.

## AI Usage Questions

1. **Did you use any AI tools (such as ChatGPT, GitHub Copilot, Cursor, or similar) while working on this assessment?**

2. **If yes, which AI tools did you use?**

3. **How did you use AI tools during this assessment?**

4. **What percentage of your code was written with AI assistance?**

5. **Did you use AI tools to help understand the requirements or project structure?**

6. **Did you use AI tools for debugging or fixing errors?**

7. **What did you learn from using AI tools (if applicable)?**

8. **If you didn't use AI tools, what resources did you use instead?**

## Decision-Making Question

**Question:** Walk us through your development process for completing this assessment. Please describe:

- How did you approach the task? What was your strategy or plan?
- What decisions did you make along the way, and why?
- Did you start with the backend or frontend first? What was your reasoning?
- How did you prioritize which features to implement first?
- What challenges did you encounter, and how did you overcome them?
- If you had more time, what would you have done differently or improved?

Please provide a detailed response (2-3 paragraphs) explaining your development process and decision-making approach.
